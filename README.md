# SaveandLoad2

《杀戮尖塔 2》存档复盘工具，把 `.run` 文件里散落的战斗记录整理成一份能直接看的对局报告。

## 功能

- **对局列表**：自动扫描存档目录，按时间排序，缩略卡片展示角色头像、升级等级、楼层、用时、Seed、胜负。
- **联机识别**：自动解析多人存档，把每位玩家的卡组、遗物、药水分开列出，并通过 SteamID 反查显示昵称。
- **逐层时间线**：每层显示进入时间、用时、HP 变化、获取/移除/升级/转化的卡牌、新增遗物、买卖事件等；行内可展开查看卡牌效果、遗物效果、药水属性，全部中文化。
- **AI 复盘**：可选填入兼容 OpenAI 的接口与模型，让大模型基于这局数据给出复盘建议。API Key 用 Windows DPAPI 加密保存在本地。
- **自动更新**：启动时静默检查新版本，发现更新时在左下角弹出按钮；设置页也可以手动检查。

## 存档位置

默认在：
```
Steam\userdata\<你的好友邀请码>\2868840\remote\profile1\saves
```

软件首次启动可在左下角"选择存档文件夹"指向该目录，之后会自动记住。
