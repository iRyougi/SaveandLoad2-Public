# SaveandLoad2 v0.3.1 (Pre-release)

> 发布日期：2026-05-31
> 类型：预发行版本（Pre-release）
> 上一版本：[v0.3.0](./RELEASE_v0.3.0.md)

本版本是一次围绕**生涯统计**的深度增强。v0.3.0 搭起了生涯仪表盘的骨架，这一版把它做成了真正能用来"复盘自己打法"的工具：可组合的筛选、按职业拆分的明细、读自游戏本体的权威累计数据，以及一套随筛选变化的卡牌收益率分析。

---

## 重点更新

### 可组合的筛选面板

生涯统计现在支持**按职业 / 难度多选组合**筛选，所有 KPI、排行、卡牌分析都会随筛选实时重算。

- 入口改为**底部居中的悬浮按钮**，点击后面板**上滑弹出**、再次点击滑回收起，开合都有顺滑动画。
- 职业、难度均可多选，任意组合（例如"只看 A20 的某职业"）。
- 职业筛选会依据你的 **SteamID 锁定本人**，多人对局里队友的职业不会混进选项。

### 职业明细表 + 总时长 / 最快获胜 KPI

新增按职业拆分的明细表，一眼看清各职业表现：

- 每个职业的**局数 / 胜率 / 平均时长 / 最快获胜**。
- 概览新增**累计总时长**与**最快获胜**两个 KPI。
- 明细以**紧凑卡片**呈现，排布更紧密、不留空档。

### 生涯档案区：读游戏本体的权威数据

新增「生涯档案」区，直接读取游戏本体的 `progress.save`，展示**建筑师伤害**与**每个职业的累计数据**。

- 数据来自游戏存档本身，是权威口径，**无需安装 mod、无需 Pro 模式**即可显示。

### 卡牌收益率分析（随筛选变化）

新增卡牌收益率分析，采用 STS1 社区常用的 **delta 口径**（持有该卡的胜率 − 当前筛选基准胜率），量化每张卡对胜率的真实贡献：

- 收益率**随筛选变化**：基于当前筛选后的 `.run` 终局卡组实时计算。
- 位于页面最底部，排在「上分卡 / 战犯卡」之下，便于从概览逐步深入。
- **上分卡 / 战犯卡**表格新增「收益」列，直接对照拿取胜率与当前基准的差值。

### 修复：所有本人向统计排除联机队友

修正了多人对局下的统计口径问题：

- **上分卡 / 战犯卡、平均承伤、平均掉血**等本人向指标，之前会把联机队友的卡牌与数据一并算入。新版本统一**只统计本人**，排除队友干扰。

### 概览新增两项指标

- **平均精英数 / 局**：平均每局击杀的精英数量。
- **卡组平均大小**：终局卡组的平均牌数。

两者同样随筛选变化，且只统计本人。

## 升级注意事项

- 从 v0.3.0 升级到 v0.3.1：直接覆盖安装即可，旧设置和数据全部保留。
- 「生涯档案」的建筑师伤害与职业累计读自游戏本体 `progress.save`，无需 mod 或 Pro 模式；若该区域为空，请确认游戏本体存档可被读取。
- 职业筛选与本人向统计依赖 **SteamID64**，建议在「设置」里填一次，以便正确区分本人与联机队友。
- 已生成的战斗日志、Pro 模式启用状态、Steam 路径配置都会保留。
