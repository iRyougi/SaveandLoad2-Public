# SaveandLoad2 v0.3.3 (Beta)

> 发布日期：2026-06-21
> 类型：测试版（Beta）
> 上一版本：[v0.3.2](./RELEASE_v0.3.2.md)

本版本是一次**功能 + 数据可信度 + 可支持性**的较大更新：新增基于战斗日志的伤害/弱化统计，补齐局内成就与遗物真实图标，从根上修复了数据采集 mod 的两个关键 bug，并引入文件日志、Pro Mode mod 自动升级等运维改进。建议所有用户更新。

---

## 新功能

### 战斗统计：伤害总结 + 弱化统计

在对局详情页新增「本局战斗统计」面板（需开启 Pro Mode 采集战斗日志）：

- 按玩家列出**整局总伤害**、**虚弱 / 易伤 / 总弱化**次数，按总伤害降序排列，联机局一目了然谁是输出/控制核心。
- 每层战斗展开后还有「本场统计」小计，与该层回合时间线对应。
- 多人局玩家名会自动把 SteamID 翻译成 Steam 昵称。

### 局内成就展示

详情页展示该局获得的**局内成就徽章**（如暴食者、治疗专家、完胜、伤害第一、弱化第一等），按档次配色，数据来自存档本身，无需 mod。

### 遗物真实图标

对局里的遗物从默认紫色钻石占位**替换为游戏原版图标**（汇总区与逐层「本层遗物」均生效），个别未收录的遗物回退占位、不报错。

### 单人 / 多人筛选

生涯统计页筛选改为三栏布局，并支持**按单人 / 多人**过滤，统计口径更清晰。

---

## 数据采集修复（Pro Mode mod）

修复了战斗日志采集的两个关键 bug，让战斗统计的数据真正可信：

- **终结一击不再丢失**：此前击杀目标的最后一击因游戏在击杀时跳过某个 hook 而未被记录，导致总伤害偏低；改挂更可靠的 hook 后，致死伤害也能正确计入。
- **存读档不再丢前半局**：同一局因存档/读档被拆进多个文件夹时，此前只读到后半段；现按「种子 + 时间窗 + 楼层不重叠」把碎片合并回同一局，前半局战斗重新出现。

> 注：以上修复对**新打的战斗**生效；升级后请重新游玩以采集到完整数据。

---

## 稳定性与可支持性

- **文件日志**：应用运行日志现写入**软件安装目录下的 `logs/app-YYYYMMDD.log`**（保留 7 天，启动即写一行版本心跳）。遇到问题时附上该日志即可快速定位。
- **Pro Mode mod 自动升级**：检测到游戏目录里的采集 mod 版本与内置版本不一致时，会在游戏未运行时**静默重装到最新**，不再需要手动「重新启用」。
- **兼容创意工坊版 BaseLib**：BaseLib 通过 Steam 创意工坊安装（不在游戏 `mods` 目录）时，现在也能正确识别，Pro Mode 不再误报「未检测到 BaseLib」。
- **启动更稳更快**：首页 / 战斗记录 / 生涯统计三页改为并行加载，并加了异常兜底，单页加载失败不再拖垮整个应用。

## 其它修复

- **选择对话框双击**：在「选择 Steam 库」等列表对话框里**双击某项 = 确定**（此前双击无效）。
- **更新公告显示**：首页「软件更新公告」现在展示当前版本的真实说明，不再显示过期的固定文案。
- 内部清理：抽取统一的 SteamID 工具、引入日志框架替换零散调试输出、README 以 UTF-8 重写。

---

## 升级注意事项

- 从 v0.3.2 升级：直接覆盖安装即可，所有设置与数据保留。
- 战斗统计需要开启 **Pro Mode** 采集战斗日志；升级后建议重新游玩以采集到本版修复后的完整数据。
- 已开启 Pro Mode 的用户：启动应用后采集 mod 会自动升级到本版，无需手动操作。
