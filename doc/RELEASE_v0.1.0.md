# SaveandLoad2 v0.1.0 (Pre-release)

> 发布日期：2026-05-21
> 类型：预发行版本（Pre-release）

《杀戮尖塔 2》存档复盘工具的首个公开预览版本。此版本已实装对局解析、详情展示、AI 复盘与自动更新等核心功能，欢迎尝鲜并反馈问题。

---

## 主要功能

### 对局列表
- 自动扫描 Steam 存档目录下的 `.run` 文件，按时间倒序排列。
- 卡片化展示：角色头像、升级等级、当前楼层、用时、Seed、胜负结果。
- 支持 5 个杀戮尖塔 2 角色头像（取自游戏内 `character_select` 与 `map_marker` 资源，按 alpha bbox 精确居中）。

### 联机存档解析
- 自动识别多人联机模式存档，分别展示每位玩家的卡组、遗物、药水。
- 通过 SteamID 反查并显示玩家昵称。

### 对局详情页
- 三段式布局：基础信息、楼层时间线、最终状态。
- 逐层展示：进入时间、用时、HP 变化、卡牌增删改/转化、遗物获取、商店买卖等事件。
- 行内可展开查看卡牌效果、遗物效果、药水属性，全部中文化（基于内置静态属性表与 `StsLocalization.DescribeCard/Relic/Potion`）。
- 详情页种子（Seed）支持点击复制，带 hover 高亮与居中复制成功浮层。

### AI 复盘
- 兼容 OpenAI 接口规范，可自定义 BaseURL 与模型名。
- 双模式支持：仅生成提示词（手动复制到外部 AI 使用）或直连 API 自动生成复盘。
- API Key 使用 Windows DPAPI 加密保存在本地，不上传任何服务器。

### 自动更新
- 启动时静默检查 GitHub `version.json`，发现新版本时左下角按钮显示更新标记。
- 设置页提供"检查更新"卡片，可手动触发检查。

### 界面与交互
- 基于 WPF-UI 重构主窗口，提供现代化外观。
- 操作按钮集中在左栏底部，设置页内容居中布局。
- 应用 Logo 与设置界面统一视觉风格。

---

## 已知问题与限制

- 仅支持 `schema_version = 9` 的 `.run` 文件，旧版本存档解析可能失败。
- 联机存档昵称解析依赖 Steam 客户端在线状态，离线时仅显示 SteamID。
- AI 复盘的提示词与 API 模式尚未做大规模真机测试，部分模型返回格式可能异常。
- 目前仅打包 Windows 版本。

---

## 安装与使用

1. 在 [Releases](https://github.com/iRyougi/SaveandLoad2-Public/releases) 下载本版本附件。
2. 解压后运行可执行文件。
3. 首次启动时在左下角点击"选择存档文件夹"，指向：
   ```
   Steam\userdata\<好友邀请码>\2868840\remote\profile1\saves
   ```
4. 之后软件会自动记住该路径。
