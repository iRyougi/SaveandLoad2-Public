# SaveandLoad2 v0.2.2 (Pre-release)

> 发布日期：2026-05-24
> 类型：预发行版本（Pre-release）
> 上一版本：[v0.2.1](./RELEASE_v0.2.1.md)

本版本重点解决「Steam 安装路径」相关的体验问题。之前 Pro 模式只能从默认位置寻找游戏，对把游戏装在其他盘或者有多个 Steam Library 的用户不友好。新版本把 Steam 路径提升为全局设置，并改善了路径识别与切换体验。

## 升级注意事项

- 从 v0.2.1 升级到 v0.2.2：直接覆盖安装即可。
- 首次启动新版本时建议到「设置 → Steam 路径」确认一下识别出的路径是否正确，特别是有多个 Steam Library 的用户。
- 已生成的战斗日志、Pro 模式启用状态都会保留，无需重新部署 Mod。

## 反馈

预发行版本难免存在问题，欢迎通过 GitHub Issues 或邮件 `iryougi@iryougi.com` 反馈。如果遇到 Steam 路径无法识别或选择窗口未弹出的情况，请附上你的 Steam 安装位置与 SteamLibrary 目录结构，便于复现。
