# SaveandLoad2 v0.3.4 (Beta)

> 发布日期：2026-06-30
> 类型：测试版（Beta）
> 上一版本：[v0.3.3](./RELEASE_v0.3.3.md)

本版本是一次**数据补全 + 体验打磨**更新：补录毒/灾厄等持续伤害并在战斗统计里独立汇总，新增战绩搜索与筛选、帮助文档入口，并对多页切换、玩家切换、启动加载做了一轮动效与流畅度优化。建议所有用户更新。

---

## 新功能

### 毒 / 灾厄等持续伤害统计

战斗统计此前只统计直接伤害，毒、灾厄这类**持续伤害（DoT）**没有归属对象、被漏算。本版本由采集 mod 补录这类伤害，并在对局详情的战斗统计里以**独立命名桶**展示：

- 玩家伤害列表下方新增「毒伤害」「灾厄伤害」分项（有数据才显示），整局与逐层「本场统计」均生效。
- 这类伤害不强行算到某个玩家头上，单独成项，口径更清晰。

> 注：该数据由本版采集 mod 记录，对**升级后新打的战斗**生效；旧日志不含这类数据，详情页不会显示对应分项。

### 战绩搜索 + 结果 / 联机筛选

战斗记录页顶部新增搜索框与快捷筛选：

- 按**角色 / 种子 / 结果 / 死因 / 上限等级 / 日期 / 玩家昵称**即时模糊搜索（空格分词，多关键词同时匹配）。
- 一行快捷筛选：全部 / 胜利 / 失败 / 放弃 + 「仅联机」，可与搜索词叠加，右上角显示「命中 / 总数」。

### 帮助文档入口

新增在线帮助文档双入口：**首页「开始游戏」上方**的「帮助文档」按钮，以及**标题栏右上角**的 `?` 图标（仿系统按钮风格），点击直达在线文档。

## 体验与流畅度

- **切页转场动画**：首页 / 战斗记录 / 生涯统计 / 设置 之间切换改为两页交叉滑动，方向跟随导航顺序，更有空间感。
- **启动加载动画**：启动时显示 Logo + 加载转圈，期间在后台完成数据加载与各页首次渲染预热；加载完成后淡出，**消除首次切页的卡顿**。
- **玩家 Tab 居中导航**：联机局详情页的玩家切换条，选中项会平滑滚动到居中，两端加渐隐提示可左右滑动，去掉了突兀的横向滚动条。

---

## 修复与改进

- **Steam 路径大小写对齐**：自动检测 / 手动选择 / 保存的 Steam 安装与库路径，统一纠正为磁盘上的真实大小写（如显示 `D:\Steam` 而非 `d:\steam`），避免同一路径出现多种写法。
- **日志占用提醒**：战斗日志累计超过 100 MB 时，首页会出现提示条，可一键跳转设置 · Pro Mode 清理，或选择「不再提示」（本版本内不再弹出）。

---

## 升级注意事项

- 从 v0.3.3 升级：直接覆盖安装即可，所有设置与数据保留。
- 毒 / 灾厄伤害统计需要开启 **Pro Mode** 采集战斗日志；升级后建议重新游玩以采集到本版新增的持续伤害数据。
- 已开启 Pro Mode 的用户：启动应用后采集 mod 会自动升级到本版，无需手动操作。
