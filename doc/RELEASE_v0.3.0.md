# SaveandLoad2 v0.3.0 (Pre-release)

> 发布日期：2026-05-28
> 类型：预发行版本（Pre-release）
> 上一版本：[v0.2.3](./RELEASE_v0.2.3.md)

本版本是一次大改版：软件从「单页面（战斗记录列表）」演进为「多页面应用」，正式有了**首页 / 生涯 / 战斗 / 设置**四块独立空间。除了页面层面的扩展，新增了生涯统计仪表盘、Steam 资料/游玩时长展示、远端新闻公告等一系列围绕"打开软件能看到什么"的内容。

---

## 重点更新

### 全新多页面架构 + 顶栏胶囊导航

软件不再是开机直奔战斗记录列表，而是切换为四个页面的应用：

- **🏠 首页**：Steam 资料、最近游玩、新闻、更新公告，一眼看到当前状态。
- **📊 生涯**：跨对局的统计仪表盘。
- **⚔️ 战斗**：原本的 `.run` 战斗记录列表与详情复盘。
- **⚙️ 设置**：所有配置项的集中管理。

页面之间通过顶栏中央的**胶囊导航**切换。切页时 View 会被缓存，回到同一页不会重新加载、不再有空白闪烁。

### 首页：Steam 资料 / 游玩时长 / 新闻 / 更新公告

需要在「设置」里填一次 SteamID64，首页就会自动拉取：

- **个人栏**：头像、昵称（缓存秒开，后台补缺），底部固定「开始游戏」按钮，直接通过 `steam://rungameid/` 启动《杀戮尖塔 2》。
- **游玩时长**：从 Steam 的 `localconfig.vdf` 读取总时长和最近两周时长，离线也能看。
- **更新公告**：版本检查与一键更新置顶展示。
- **新闻区**：拉取远端 `news.json` 展示游戏本体相关资讯。首次加载用本地缓存秒开，再静默刷新；分段展示更易扫读。
- **新闻读不到时**：显示一个可爱的动图占位 +「当前新闻正在施工中」文案，不留死板的空白。

### 生涯统计仪表盘

新增生涯统计页，把多次对局聚合成一张可视化报表：

- **顶部 KPI 行**：胜率（含放弃局）、最高连胜、A10 胜率、平均承伤、平均时长。
- **平均掉血排行**：哪些敌人对你伤害最大。
- **致命死因排行**：你最容易被谁送走。
- **上分卡 / 战犯卡**：按"拿取胜率"排序，自动筛掉样本太少的卡（见过 ≥5 局且拿过 ≥2 局），告诉你哪些卡真的赢、哪些卡其实在拖后腿。
- **空态友好**：还没有对局数据时给出明确提示，不会留出一堆 0。

> 生涯统计与战斗记录共用同一份扫盘结果，不会因为切页就重复读盘。

### 战斗页：`.run` 解析结果持久化缓存

战斗记录之前每次进入都会重新解析所有 `.run` 文件，对局多了之后明显变慢。新版本把解析结果持久化到本地缓存，再次进入秒开；只有新出现或被修改过的 `.run` 才会重新解析。

### 设置页：左侧分类导航

设置项随着功能增多越来越多，原来的长列表不好找。新版本把设置页改为「左侧分类导航 + 右侧详情」结构，按类别分组，找选项更直观。同时把战斗记录页底部那个冗余的"设置"按钮去掉了——顶栏已经覆盖这个入口。

## 升级注意事项

- 从 v0.2.3 升级到 v0.3.0：直接覆盖安装即可，旧设置和数据全部保留。
- 首次启动新版本时，建议到「设置」里填一次 **SteamID64**，否则首页的头像、昵称、游玩时长无法显示。
- 已生成的战斗日志、Pro 模式启用状态、Steam 路径配置都会保留。
- 首次进入战斗记录页时会建立解析缓存，可能比平时稍慢；之后再开就是秒开。
